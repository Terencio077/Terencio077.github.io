Componentes CSS Melhorados

1. Breadcrumb

Início > Receitas > Macarronada

✓ Mostra localização atual no app | Melhora navegação

2. Botões com Estados

✓ Hover: destacado | Active: pressionado | Disabled: desabilitado

3. Notificações

✓ Feedback visual | Desaparece automaticamente | Não interrompe fluxo

4. Estados de Link

Link normal | Link ativo | Link página atual

✓ Feedback ao passar mouse | Cores consistentes

5. Footer com Página Ativa

✓ Item ativo tem classe "ativo" e atributo aria-current="page"
✓ Adicionado em: index.html, aba_de_pesquisa.html, calendario_footer.html

📋 Como Usar

1. Breadcrumb 
<nav aria-label="breadcrumb">
    <a href="index.html">Início</a> > 
    <span>Página Atual</span>
</nav>
2. Botão Primário 
<button class="primario">Ação Principal</button>
3. Notificação 
<!-- Adicione no final do <body> -->
<div class="notificacao">✓ Ação realizada!</div>

<!-- Com erro -->
<div class="notificacao erro">❌ Erro ao salvar</div>

<!-- Com alerta -->
<div class="notificacao alerta">⚠️ Atenção!</div>
4. Página Ativa 
<footer>
    <ul>
        <li><a href="index.html" class="ativo" 
               aria-current="page">
            <img src="img/casa.png" alt="Início">
        </a></li>
    </ul>
</footer>
